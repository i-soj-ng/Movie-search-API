# Movie-search-APP
yts 영화검색 API를 이용해 영화검색App을 제작한 후, heroku를 이용해 배포하였다.

https://movie-app-20203117.herokuapp.com/build/movieApp.html

1. 검색창에 "like 5"를 검색하면 좋아요 수가 가장 많은 5개의 영화 포스터가 뜬다.
2. 포스터를 클릭하면 각 영화의 정보를 볼 수 있다.
